# Cadastros
Projeto de estudo sobre CRUD, tem como foco de modo simples entender como fazer um CRUD funcional.

## Tecnológias
- REACT
- AXIOS
- REACT-ROUTER-DOM

## Pré-requisitos
- Node
- Npm

## Instalação
- git clone https://github.com/RCA04/cadastros.git
- cd cadastros
- npm install
- npm start

### Backend

## Pré-requisitos
- PHP V8+
- Composer

## Instalação
- git clone https://github.com/RCA04/cadastrosApi.git
- cd cadastrosApi
- composer install
- php artisan migrate
- php artisan serve
